# 💳 Digital Banking UI

A modern and responsive **Digital Banking Platform UI** built using **React.js** and **Tailwind CSS**. This project showcases a clean fintech-inspired interface with customer segmentation cards, a responsive navigation bar, and a modern user experience.

---

## 🚀 Features

- Modern and responsive UI
- Clean fintech-inspired design
- Built with reusable React components
- Customer segmentation cards
- Responsive navigation bar
- Tailwind CSS utility-first styling
- Mobile-friendly layout
- Smooth and minimal user interface

---

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- JavaScript (ES6+)
- HTML5
- CSS3
- Remix Icons
- Vite

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/OmkarMete/digital-banking-ui.git
```

Go to the project directory

```bash
cd digital-banking-ui
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
